Base de Datos:

MS SQL Server

En el folder "Base de datos" se encuentra un archivo .bak el cual contiene las tablas de la base de datos

pasos:

1- Entrar a Microsoft SQL server Management studio (instalar en caso de no tenerlo)
2- click derecho en el folder Databases (Basededatos)
3- Restore Database
4- en el apartado suurce elegir divice, agregar el archivo .bak
5- Click Ok, luego a Ok.
Listo!

Isntalador:
